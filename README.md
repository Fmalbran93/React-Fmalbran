## Pre Requerimientos

_Instalar React y las siguientes dependencias_

```
- npm i firebase
- npm i react-toastify 
- npm i react-router-dom

```

## Ejecutando la aplicacion

_Para ejecutar la aplicacion agregue el siguiente comando en la terminal_*

- npm run start


***_Luego vaya al:_ ***
```
localhost: 3000

```
